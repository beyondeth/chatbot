# 메신저봇R v2 - 구조화된 챗봇

## 📁 폴더 구조
```
bot-v2/
├── main.js           # 메인 봇 스크립트
├── config.js         # 설정 파일 (추후 분리)
├── commands/         # 명령어 모듈 (추후 분리)
│   ├── game.js      # 게임 관련 명령어
│   ├── info.js      # 정보 조회 명령어
│   └── admin.js     # 관리자 명령어
└── utils/           # 유틸리티 함수 (추후 분리)
```

## 🚀 사용 방법

1. **서버 URL 설정**
   ```javascript
   const Config = {
     SERVER_URL: "http://192.168.123.138:3001", // 실제 서버 주소로 변경
   };
   ```

2. **메신저봇R 설정**
   - 메신저봇R 앱에서 새 봇 생성
   - `main.js` 내용 복사하여 붙여넣기
   - 봇 활성화

3. **테스트**
   - `!테스트` - 서버 연결 확인
   - `!가위바위보 가위` - 게임 테스트
   - `!도움말` - 명령어 목록

## 📝 명령어 추가 방법

새로운 명령어를 추가하려면 Commands 객체에 추가:

```javascript
Commands.myCommand = {
  pattern: /^!내명령어$/,
  
  execute: function(room, msg, sender, replier) {
    // 명령어 로직
    replier.reply("응답 메시지");
  }
};
```
